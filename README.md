# Stock-Trading-Platform-
This is a stock trading platform where user can buy and sell stocks

## Steps with which I made this platform
i> Analysis of website design
ii> Breaking the website into components on basic of feature
iii> creating folders for each major componet and corresponding files for each nested minor related corresponding components
iv> structuring and ordering each minor components(with h1) of one major component into the wraping of major compopent 
v> react-router-dom : was used to make the the movement to different section of the page possible by importing { BrowserRouter, Routes, Route}
vi> Routers and No Found Page Setup
vii> Individual section completion
